                    30 Juni 2025
                

            

                    

                                    
                                

                    Mata kuliah ini memberikan

pemahaman mendalam dan keterampilan praktis yang dibutuhkan untuk menjadi

seorang administrator server Linux yang kompeten. Pembahasan mencakup konsep

dasar, arsitektur, instalasi, dan konfigurasi sistem operasi Linux. Mahasiswa

akan belajar mengelola sistem melalui antarmuka baris perintah (CLI), manajemen

pengguna dan grup, pengaturan hak akses file, manajemen paket perangkat lunak,

serta konfigurasi layanan jaringan esensial seperti web server, DNS, dan DHCP.

Penekanan utama diberikan pada otomatisasi tugas melalui shell scripting dan

penerapan dasar-dasar keamanan sistem (system hardening).
                

            

                    

                                    
                                

                    
This course provides in-depth understanding and practical skills needed to become a competent Linux server administrator. Topics covered include basic concepts, architecture, installation, and configuration of the Linux operating system. Students will learn to manage systems through the command-line interface (CLI), user and group management, file access permissions, software package management, and configuration of essential network services such as web servers, DNS, and DHCP. The primary focus is on automating tasks through shell scripting and implementing basic system security measures (system hardening)
                

            

                    

                                    
                                

                    
  • Mampu menerapkan  alat dan metode pengembangan sistem informasi yang relevan untuk membuat rekomendasi analisis bisnis organisasi, dan menilai proses sistem.
  • Mampu merancang infrastruktur SI untuk memastikan keberlanjutan operasional dan mendukung kebutuhan organisasi secara optimal.
  • Mampu membangun perangkat lunak yang dapat mendukung proses bisnis organisasi
  • Mampu menganalisis risiko terhadap aset sistem informasi dengan memprioritaskan prinsip-prinsip keamanan informasi secara efektif dan bertanggung jawab

                

            

                    

                                    
                                

                    
a.	Pengenalan Sistem Linux, Instalasi, dan Konfigurasi Dasar
b.	Dasar Command Line & Manajemen File System
c.	Manajemen Sistem Harian: Pengguna, Paket, dan Proses 
d.	Dasar-dasar Shell Scripting (Bash)
e.	Pembangunan Layanan Jaringan
f.	Proteksi Server (Hardening & Firewall)
g.	Monitoring dan Analisis Log Server
h.	Otomatisasi Manajemen Server (Pengenalan Ansible)
i.	Pengantar Containerization dengan Docker
j.	Pengantar Continuous Integration/Deployment (CI/CD)
k.	Studi Kasus: Deployment Aplikasi (Tradisional vs. Kontainer)
l.	Proyek Akhir: Presentasi dan Demo Infrastruktur Server
